One-Dimensional Search Algorithm

One-Dimensional Search Algorithm#

  1. Establish an error tolerance \(ε\). Determine an \(x_u\) such that \(df(x_u)/dx ≤ 0\) and an \(x_l\) such that \(df(x_l)/dx ≥ 0\).

  2. Compute a new trial solution \(x = (x_u + x_l)/2\).

  3. If \(x_u – x_l ≤ ε\), then terminate. The current approximation is within the established error tolerance of \(x^*\).

  4. If \(df(x)/dx ≥ 0\), set \(x_l = x\).

  5. If \(df(x)/dx ≤ 0\), set \(x_u = x\).

  6. Go to Step 2

    Example 5.2

    The algorithm can be illustrated by the problem of maximizing

    \[f(x) = x^4 - 16x^3 + 91x^2 - 216x + 180\]

    over the range \(3.2 \leq x \leq 5.0\), which is shown in Figure 5.7. The function is certainly concave in the range \(3.2 \leq x \leq 5.0\), so we will apply the search to that range.

    The derivative \(df(x)/dx = 4x^3 - 48x^2 + 182x - 216\) will be used during the procedure.

    \(\text{1. }x_u = 5.0\), \(x_l = 3.2\), and let \(\epsilon = 0.15\)
    \(\text{2. }x = (5.0 + 3.2)/2 = 4.1\)
    \(\text{3. }5.0 - 3.2 = 1.8 > \epsilon\)
    \(\text{4. }df(x)/dx\) at \(x = 4.1\) is equal to -0.996 < 0, so set \(x_u = 4.1\) and leave \(x_l = 3.2\)
    \(\text{2. } x = 3.65\)
    \(\text{3. }4.1 - 3.2 = 0.9 > \epsilon\)
    \(\text{4. }df(x)/dx\) at \(x = 3.65\) is equal to 3.328 > 0, so set \(x_l = 3.65\) and leave \(x_u = 4.1\)
    \(\text{2. }x = 3.875\)
    \(\text{3. }4.1 - 3.65 = 0.45 > \epsilon\)
    \(\text{4. }df(x)/dx\) at \(x = 3.875\) is equal to 1.242 > 0, so set \(x_l = 3.875\) and leave \(x_u = 4.1\)
    \(\text{2. }x = 3.988\)
    \(\text{3. }4.1 - 3.875 = 0.225 > \epsilon\)

    FIGURE 5.7
    \(f(x) = x4 – 16x3 + 91x2 – 216x + 180\).

    1. \(df(x)/dx\) at \(x = 3.988\) is equal to \(0.12 > 0\), so set \(x_l = 3.988\) and leave \(x_u = 4.1\)

    2. \(x = 4.044\)

    3. \(4.1 - 3.988 = 0.112 < \epsilon\), so the process terminates with the current trial solution \(x = 4.044\) and a function value of 3.99

    Notice that at the point \(x = 4.044\), the derivative of \(f(x)\) is \(-0.44\) (close to zero); and at the true optimum of \(x = 4\), where \(f(x) = 4\), the derivative is exactly zero, a necessary condition for optimality.

    Other methods for a one-dimensional search include the Fibonacci method and a related technique called the golden section method. These methods are discussed and compared in Wilde (1964). The golden section search is based strictly on the use of function evaluations, and is particularly useful when first derivatives are not available. (See Exercise 5.6.)

    While a single-variable search method may seem too simplistic for practical nonlinear optimization problems, such methods are often incorporated into more elaborate multivariate search procedures, and therefore warrant our awareness and understanding.

import numpy as np
import matplotlib.pyplot as plt

# Function and derivative
def f(x):
    return x**4 - 16*x**3 + 91*x**2 - 216*x + 180

def df(x):
    return 4*x**3 - 48*x**2 + 182*x - 216

def bisection_method_for_max(df, xl, xu, eps):
    """Bisection method applied to derivative to find maximum of f(x)"""
    iter_data = []
    print(f"{'Iter':>4} | {'x_l':>8} | {'x_u':>8} | {'x_r':>8} | {'f\'(x_l)':>10} | {'f\'(x_r)':>10} | Width")
    print("-"*70)

    while (xu - xl) > eps:
        xr = (xl + xu) / 2
        f_l = df(xl)
        f_r = df(xr)

        iter_data.append((xl, xu, xr, f_l, f_r))
        print(f"{len(iter_data):4d} | {xl:8.4f} | {xu:8.4f} | {xr:8.4f} | {f_l:10.4f} | {f_r:10.4f} | {xu - xl:8.4f}")

        if f_l * f_r < 0:
            xu = xr
        else:
            xl = xr

    xr_final = (xl + xu) / 2
    return xr_final, iter_data

def plot_iterations(f, iter_data, x_opt, xl_init, xu_init):
    x_vals = np.linspace(xl_init - 0.2, xu_init + 0.2, 400)
    y_vals = f(x_vals)

    plt.figure(figsize=(8,5))
    plt.plot(x_vals, y_vals, label="f(x)", color="blue")

    # Initial bounds
    plt.axvline(xl_init, color="green", linestyle="--", label=f"Initial x_l={xl_init}")
    plt.axvline(xu_init, color="purple", linestyle="--", label=f"Initial x_u={xu_init}")

    # Midpoints from iterations
    mids = [step[2] for step in iter_data]
    mids_y = [f(m) for m in mids]
    plt.scatter(mids, mids_y, color="orange", zorder=5, label="Midpoints")

    # Label each midpoint with iteration number
    for i, (m, y) in enumerate(zip(mids, mids_y), 1):
        plt.text(m, y+0.3, str(i), fontsize=8, ha='center', color="darkred")

    # Final optimum
    plt.scatter(x_opt, f(x_opt), color="red", s=100, zorder=6, label=f"Optimum ~ {x_opt:.4f}")
    plt.axvline(x_opt, color="red", linestyle="--")

    plt.xlabel("x")
    plt.ylabel("f(x)")
    plt.title("Bisection Method on f'(x) to Find Maximum of f(x)")
    plt.legend()
    plt.grid(True)
    plt.show()

# --- Main execution ---
if __name__ == "__main__":
    xl_init = 3.2
    xu_init = 5.0
    eps = 0.15

    x_opt, data = bisection_method_for_max(df, xl_init, xu_init, eps)
    f_opt = f(x_opt)

    print("\nFinal Result:")
    print(f"x_opt ≈ {x_opt:.5f}")
    print(f"f(x_opt) ≈ {f_opt:.5f}")

    plot_iterations(f, data, x_opt, xl_init, xu_init)
